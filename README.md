تطبيق ويب شامل لتحليل الأسهم السعودية مع جلب البيانات المباشرة من السوق السعودية وتوقعات الخبراء.

## 🚀 البدء السريع

### متطلبات التشغيل
- Python 3.8 أو أحدث
- متصفح Chrome (لعمل web scraping)
- اتصال بالإنترنت

### التثبيت
```bash
# استنساخ المشروع
git clone https://github.com/YOUR_USERNAME/saudi-stock-analyzer.git
cd saudi-stock-analyzer

# إنشاء بيئة افتراضية
python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # Linux/Mac

# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل التطبيق
python stock_web_app.py
```

ثم اذهب إلى: http://localhost:5000

## 🎮 كيفية الاستخدام

### 1. إضافة الأسهم
- أدخل رمز السهم (مثال: 1210)
- أدخل اسم الشركة (مثال: الراجحي)
- اضغط "إضافة سهم آخر" للمزيد

### 2. استخدام الأسهم الشائعة
- اضغط "تحميل الأسهم الشائعة"
- انقر على أي سهم لإضافته للقائمة

### 3. تشغيل التحليل
- اضغط "تحليل الأسهم"
- انتظر النتائج (قد تستغرق 1-2 دقيقة)

### 4. قراءة النتائج
- **السعر الحالي**: آخر سعر من السوق
- **السعر الأول**: أول سعر مسجل
- **التغير**: الفرق والنسبة المئوية
- **السعر المستهدف**: توقع الخبراء
- **العائد المحتمل**: النسبة المئوية المتوقعة
- **التوصية**: شراء/بيع/احتفاظ

## 📊 الأسهم المدعومة

التطبيق يدعم جميع الأسهم المدرجة في السوق السعودية، مع تركيز خاص على:
